#### 简单性能报告

机器参数

- 执行稽核程序的机器参数为：
  1. cpu: 8核
  2. men: 16GB
  3. net：内网，不统计
  
- presto服务器参数：
  - cpu：4核
  - mem：15G
  - net：内网，不统计
  - 1个worker节点


---

presto 内存配置 

**jvm.config：**

```shell
-Xmx10G
```

**config.properties：**

```shell
query.max-memory=8GB
query.max-memory-per-node=5GB
query.max-total-memory-per-node=6GB
```
